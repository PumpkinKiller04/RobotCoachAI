书本元素 × 磨砂玻璃：三种结合方式

背景是全屏动画（乐高机器人在学校操场→城市高楼间跑步跨栏），前景是登录入口

A

「翻开的立体书」嵌入磨砂玻璃

画面中央一本打开的立体书悬浮在动画背景之上，书页本身是磨砂玻璃材质，登录表单写在书页上。翻书动画作为页面入场效果。

关键词：立体书 + 磨砂玻璃书页 + 翻书入场

B

「书本相框」—— 书本作为磨砂玻璃的边框

画面中央是一块大面积的磨砂玻璃卡片，四周被打开的书本边框包裹——书脊在左、翻开的书页纹理环绕卡片边缘。登录入口在磨砂玻璃上。书本不是主体，而是"画框"。

关键词：书本边框 + 磨砂玻璃主体 + 书脊装饰

C

「书页飘落」—— 磨砂玻璃卡片 + 散落的书页动画

磨砂玻璃卡片居中独立存在，周围有像素风格的书页碎片缓缓飘落（像落叶一样）。登录入口在卡片上。书本不作为一个整体出现，而是"解构"成飘散的元素。

关键词：解构书页 + 飘落动画 + 磨砂玻璃独立卡片
