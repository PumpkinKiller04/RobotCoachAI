三种设计方向：登录 → 课程 的连贯书本体验

每个方向都包含登录页（左）和课程页（右）的视觉草图

A. 「桌面上的魔法书」

视角是一个学生书桌。桌面上摊开一本大书。背后的窗户望出去是乐高机器人跑步的世界。书页本身有磨砂玻璃质感，登录表单在书页上。课程页就是这本书的内部——翻页效果自然连贯。

登录页：桌面 + 窗外动画 + 书本登录
🤖
课程页：书本展开全屏 + 书页内容区

优点

  • 桌面+窗户的纵深空间感强，画面层次丰富
  • 从"桌面的书"到"书的内容"很自然
  • 窗外动画独立播放，不干扰登录操作

缺点

  • 桌面场景绘制成本高，需要较多视觉素材
  • 移动端桌面透视可能不好适配

B. 「展开的巨型书本」

整个画面就是一左一右翻开的巨大书本。左页是登录区（磨砂玻璃质感覆盖在书页上），右页嵌入机器人动画小窗。课程页就是书本自然翻到内容页——左右两页分别是课程导航和内容。

登录页：左页登录 + 右页动画窗
机器人总教练
🤖
从"醉汉走路"到跑赢人类
课程页：书本内容自然展开

优点

  • 书本隐喻最直观、最纯粹
  • 左右双页结构天然适配双栏布局
  • 登录→课程过渡只需"翻页"动画

缺点

  • 纯书本视觉可能略显单调
  • 右页动画区域在登录页上可能利用率不高

C. 「书本即世界」（更奇幻的路线）

书本本身就是"地面"——书页是大地，文字变成跑道，机器人在书本的字里行间跑步。登录卡片悬浮在场景上方。课程页则是完全沉浸在书的世界里，内容直接在书页纹理上呈现。

登录页：书本地面 + 机器人奔跑 + 悬浮卡片
🤖🏃‍♂️📖
课程页：沉浸书页世界

优点

  • 最独特、最有记忆点
  • 书本和机器人动画融合得最好
  • 沉浸感最强

缺点

  • 实现复杂度最高
  • 奇幻风格可能不如写实风格"稳重"
  • 可读性需要在设计上更小心处理
