# AI总教练指挥中心 - 微课网站

## 项目概览

这是一个为学生设计的交互式微课网站，配套教材讲解"多模态感知"概念。网站包含四个学习环节，通过动画、交互式演示，帮助学生理解机器人如何通过多种传感器获取和整合信息。

## 技术特点

- **纯静态实现**：无需任何服务器或构建工具
- **单文件部署**：所有功能集成在 index.html 中
- **无外部依赖**：无需安装 npm 包或配置环境
- **跨平台兼容**：支持所有现代浏览器
- **离线可用**：下载后可直接使用

## 文件结构

```
├── index.html      # 主页面（包含所有功能）
├── README.md       # 使用说明
└── public/         # 静态资源目录（可扩展）
```

## 功能模块

### 标签页1: 选手档案
- 机器人线框图扫描动画
- 模拟试跑功能（第一视角 + 上帝视角）
- 传感器状态显示
- 试跑失败分析反馈

### 标签页2: 教练改装车间
- 拖拽传感器安装交互
- 三次递进式试跑体验
- 数据层可视化
- 安装进度条

### 标签页3: 感官大发现
- 人形机器人SVG插图
- 4个可点击热点（摄像头、激光雷达、IMU、力觉传感器）
- 发现进度追踪
- 完成时光晕动画

### 标签页4: 多模态体验站
- 图片上传功能（拖拽/点击）
- 模拟AI响应演示
- 热门问题快捷按钮
- 前沿应用展示卡片

## 使用方式

双击 `index.html` 在浏览器中打开即可使用。

## 浏览器兼容性

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## 注意事项

此版本为纯静态版本，所有AI功能通过预设模板模拟实现，适合在无网络环境或比赛中使用。
