# AI总教练指挥中心 - 多模态感知微课

一个纯静态的交互式微课网站，配套"多模态感知"教学内容。

## 内容结构

网站包含四个学习环节：

| 标签页 | 名称 | 功能 |
|--------|------|------|
| 1 | 选手档案 | 机器人线框图 + 模拟试跑 |
| 2 | 教练改装车间 | 拖拽传感器升级 |
| 3 | 感官大发现 | 点击热点学习传感器 |
| 4 | 多模态体验站 | 图片上传 + Kimi AI问答 |

## 使用方法

双击 `index.html` 在浏览器中打开即可使用。

## 多模态体验站说明

第四个标签页集成了**硅基流动 Kimi-K2.6** 多模态大模型，支持：
- 图片理解（上传图片后提问）
- 文字对话
- 流式响应效果

**重要提示**：由于浏览器安全策略，从文件系统打开时需要通过CORS代理访问API。

## 技术特点

- 纯静态 HTML + CSS + JavaScript
- 无需构建工具
- 支持暗色模式（跟随系统）
- 响应式设计
- 集成硅基流动多模态API

## 文件大小

- `index.html`: ~45KB（单文件包含所有功能）
